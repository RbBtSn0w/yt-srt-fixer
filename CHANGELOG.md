## [1.0.0] - 2025-12-29
- Initial public release of FCP SRT Fixer.
- Fixes overlapping SRT timestamps with a configurable minimum gap.
- Supports processing single files or entire directories.
- Generates cleaned subtitles with the `_fcp.srt` suffix.
- Adds open-source scaffolding: README, CONTRIBUTING, CODE_OF_CONDUCT, LICENSE.
